# 💈 Gestión Barbería - API Backend

Sistema de gestión integral para barberías desarrollado con FastAPI y PostgreSQL.

## 🎯 Características

- Gestión de usuarios (clientes, barberos, administradores)
- Sistema de reservas de turnos
- Catálogo de servicios
- Autenticación JWT
- Arquitectura limpia

## 🛠️ Stack Tecnológico

- **Backend:** FastAPI
- **Base de datos:** PostgreSQL
- **ORM:** SQLAlchemy
- **Testing:** pytest
- **Containerización:** Docker
