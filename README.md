# Barbería API

API REST para gestión de turnos y servicios de barbería.

## Stack

FastAPI · PostgreSQL · SQLAlchemy · Pydantic · Docker

## Estado actual

Proyecto en desarrollo activo. La estructura base está implementada, trabajando ahora en los endpoints y la capa de autenticación.

**Completado:**
- Modelos de datos (usuarios, barberos, servicios, turnos)
- Configuración de base de datos con SQLAlchemy
- Schemas de validación

**Próximos pasos:**
- Endpoints CRUD para cada entidad
- Sistema de autenticación con JWT
- Tests unitarios y de integración
- Dockerización completa
